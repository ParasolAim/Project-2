# Block-Based Video Transform & Quantization Codec

## 1. Environment Setup
* **Software Requirement:** Installing MATLAB 

## 2. Source Code Execution
1. Place the video file in the project folder.
2. Open `final.m` in MATLAB.
3. Configure your `QP` and `blockSize` parameters at the top of the script.
4. Change the current path in the video file of the code to yours. 
5. Click "Run" to execute the pipeline, view metrics charts, and play the output video.
